Chapter 3: Coordinate Geometry

Topics Covered:

  • Introduction to Coordinate Plane
  • Cartesian System
  • Plotting Points
  • Quadrants

1. Introduction to Coordinate Geometry

Coordinate geometry is the branch of mathematics in which geometric problems are solved using coordinate systems and algebraic techniques.

2. Cartesian System

The Cartesian system consists of two perpendicular lines:

  • x-axis: The horizontal line
  • y-axis: The vertical line

The point where these two axes intersect is called the origin (0, 0).

3. Coordinates of a Point

Coordinates are written as an ordered pair (x, y) where:

  • x: Distance from the y-axis (horizontal distance)
  • y: Distance from the x-axis (vertical distance)

4. Quadrants

The x- and y-axes divide the plane into four regions called quadrants:

  • Quadrant I: (+x, +y)
  • Quadrant II: (−x, +y)
  • Quadrant III: (−x, −y)
  • Quadrant IV: (+x, −y)

5. Plotting Points

To plot a point, start from the origin:

  • Move right/left for x
  • Move up/down for y

Example: To plot (3, 2), move 3 units right and 2 units up.

Conclusion

Coordinate geometry helps us represent and solve geometric problems in a two-dimensional space. Understanding how to read and plot points is the first step.
