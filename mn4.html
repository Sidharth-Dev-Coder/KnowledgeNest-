Chapter 4: Linear Equations in Two Variables

Introduction

This chapter deals with equations of the form ax + by + c = 0, where a, b, and c are real numbers and a and b are not both zero.

1. Linear Equation

A linear equation in two variables is an equation which can be written in the form:

ax + by + c = 0

Examples: 2x + 3y = 6, x – y = 4

2. Solutions of Linear Equations

  • A solution is a pair of values (x, y) that satisfies the equation.
  • There are infinitely many solutions to a linear equation in two variables.
  • Example: x + y = 5 ⇒ (2, 3), (1, 4), (0, 5) all are solutions.

3. Graph of a Linear Equation

  • The graph of a linear equation in two variables is a straight line.
  • To draw the graph, plot at least two solutions and join them with a straight line.
  • Example: For x + y = 5, use (0,5), (5,0), (2,3)

4. Important Notes

  • Every point on the line is a solution of the equation.
  • If a point does not lie on the line, it is not a solution.
  • If we fix the value of x, we get a unique value of y, and vice versa.

5. Applications

  • Used in solving real-life problems where two quantities vary linearly.
  • Example: Cost of items, time-distance problems, etc.

Conclusion

Linear equations in two variables provide a strong foundation for algebra and coordinate geometry. Understanding how to interpret and graph them is essential for solving problems effectively.
