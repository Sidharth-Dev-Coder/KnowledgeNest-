Chapter 2: Polynomials

Topics Covered:

  • Polynomials and Algebraic Expressions
  • Degree of Polynomial

  • Addition, Subtraction, and Multiplication of Polynomials
  • Zeroes of a Polynomial

1. What is a Polynomial?

A polynomial is an algebraic expression made up of one or more terms, where each term includes a variable raised to a whole number exponent.

Example: 2x² + 3x - 5

2. Degree of a Polynomial

The highest power of the variable in a polynomial is called its degree.

Example: In 4x³ + 2x² + 1, the degree is 3.

3. Types of Polynomials

  • Monomial: A polynomial with one term (e.g., 5x)
  • Binomial: A polynomial with two terms (e.g., x + 2)
  • Trinomial: A polynomial with three terms (e.g., x² + 3x + 2)

4. Zero of a Polynomial

Any value of x for which the polynomial becomes 0 is called a zero or root of the polynomial.

If p(x) = 2x - 4, then x = 2 is a zero because p(2) = 0.

5. Remainder Theorem

If p(x) is divided by (x – a), then the remainder is p(a).

Example: If p(x) = x² - 3x + 2, then remainder when divided by x - 1 is p(1) = 0.

6. Factor Theorem

If p(a) = 0, then (x – a) is a factor of p(x).

This theorem helps in factorizing polynomials.

7. Factorization of Polynomials

Polynomials can be factored using methods like:

  • Taking common factors
  • Using identities: (a + b)², a² – b², etc.
  • Splitting the middle term

8. Algebraic Identities Used

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a² - b² = (a + b)(a - b)
  • (x + a)(x + b) = x² + (a + b)x + ab

Conclusion

Understanding polynomials is essential as it lays the groundwork for algebra. Mastering the concepts of zeroes, factorization, and algebraic identities is crucial.
